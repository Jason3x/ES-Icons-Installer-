## ES I os Installer
![Platform](https://img.shields.io/badge/Platform-R36S-blue)
![OS](https://img.shields.io/badge/OS-%20dArkOS%20RE%20|%20darkos4clone-green)
![Shell](https://img.shields.io/badge/Bash-Script-yellow)
![License](https://img.shields.io/badge/License-Free-lightgrey)

ES Icons Installer is a Bash script designed for darkos4clone and dArkOS RE that allows easy to add Wifi and Bluetooth icons in emulationsation.


---

- **5-state WiFi icon** with distinct colors:<br>
  - <img src="https://img.shields.io/badge/-WiFi-red?style=flat-square&logo=wifi&logoColor=white"> → Disabled / rfkill blocked
  - <img src="https://img.shields.io/badge/-WiFi-orange?style=flat-square&logo=wifi&logoColor=white"> → Interface up, no IP
  - <img src="https://img.shields.io/badge/-WiFi-green?style=flat-square&logo=wifi&logoColor=white"> → Connected
  - <img src="https://img.shields.io/badge/-Sharing-gold?style=flat-square&logo=gitbook&logoColor=white"> → Sharing services active (SSH / Samba / Filebrowser)
  - <img src="https://img.shields.io/badge/-Active-cyan?style=flat-square&logo=cloudup&logoColor=white"> → Active client connected to a sharing service

- **3-state Bluetooth icon**:
  - <img src="https://img.shields.io/badge/-Bluetooth-red?style=flat-square&logo=bluetooth&logoColor=white"> → Disabled
  - <img src="https://img.shields.io/badge/-Bluetooth-orange?style=flat-square&logo=bluetooth&logoColor=white"> → Active, no device connected
  - <img src="https://img.shields.io/badge/-Bluetooth-blue?style=flat-square&logo=bluetooth&logoColor=white"> → Device connected

- Toggle switches for WiFi and Bluetooth icons in **START > UI SETTINGS**
- Refresh rate: **Instantly,as soon as there is a change of state**
- ScreenScraper key support 
- Automatic binary backup before deployment

## 📋 Requirements / Important notes
- Internet connection (first-time install only)
- Based on `christianhaitian/EmulationStation-fcamod` and  `lcdyk0517/EmulationStation-fcamod`

---

## 🚀 Installation

1. Download the `ES Icons Installer` script.
2. Copy it to: `roms/tools`
3. Launch it from the **Tools** section on your device.
